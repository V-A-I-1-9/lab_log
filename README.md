## Getting Started

To get a local copy up and running, follow these simple steps.

1.  Clone the repo
    ```sh
    git clone [https://github.com/your_username/lab_log.git](https://github.com/your_username/lab_log.git)
    ```
2.  Install packages
    ```sh
    flutter pub get
    ```
3.  Run the app
    ```sh
    flutter run
    ```
